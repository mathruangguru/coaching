Statistika: Nilai Rata-rata (Mean)
Bagian A: Rangkuman Materi

1. Konsep Dasar Rata-rata (Mean)

Rata-rata (\(\bar{x}\)) dari suatu kumpulan data adalah jumlah seluruh nilai data dibagi dengan banyaknya data.

\[\bar{x} = \frac{x_1 + x_2 + x_3 + \dots + x_n}{n} = \frac{\sum x}{n}\]

Dari rumus di atas, kita juga dapat mencari Jumlah Total Data jika rata-rata dan banyaknya data diketahui:

\[\text{Jumlah Total Data} = n \times \bar{x}\]

2. Rata-rata Gabungan

Jika terdapat dua kelompok data atau lebih yang digabungkan, rata-rata gabungannya dapat dicari dengan menjumlahkan total nilai semua kelompok dibagi dengan total frekuensinya.

\[\bar{x}_{gab} = \frac{n_1 \cdot \bar{x}_1 + n_2 \cdot \bar{x}_2}{n_1 + n_2}\]

3. Aljabar pada Rata-rata (Tipe Soal Analisis/HOTS)

Seringkali pada soal tingkat lanjut, suatu nilai data didefinisikan berdasarkan nilai rata-rata dari seluruh data itu sendiri (termasuk nilai yang belum diketahui tersebut).

Contoh Pendekatan:
Jika diketahui data \(A, B, C\), dan \(D\). Nilai \(D\) adalah \(k\) lebih besar dari rata-rata keempat data tersebut. Maka persamaannya disusun sebagai berikut:

\[D = \bar{x} + k \implies D = \left( \frac{A + B + C + D}{4} \right) + k\]

Untuk menyelesaikan persamaan ini, kalikan kedua ruas dengan 4, kemudian kumpulkan variabel \(D\) di satu sisi untuk mencari nilainya.
